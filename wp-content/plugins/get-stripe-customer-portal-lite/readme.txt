Send your clients links to their Stripe Customer Portal.

== Description ==
With <strong>Get Stripe Customer Portal Light</strong> WordPress plugin your customers can get email with a link to their Stripe Customer Portal. No styles are enforced, so you can style it to look whatever your want. This is the light version with non-customizable labels and messages. Upgrade to Pro version for more flexibility and features.

= Features =
* Provides short code to be insterted anywhere on your site.
* Test and live Stripe modes

`[get_stripe_customer_portal]`
